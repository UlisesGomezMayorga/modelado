Curso de Modelado e Impresión 3D

Actividades

Bienvenid@ a la web donde se va a subir el material del curso de modelado e impresión 3D.

Este curso propone plantear situaciones problemáticas que propicien la construcción de conocimientos y saberes sobre los siguientes aspectos:

  • Manejo de herramientas de software para el modelado 3D
  • Manejo de herramientas de software para la impresión 3D
  • Exportar a formatos compatibles de impresión 3D
  • Uso de programas para la configuración de parámetros de la impresora
  • Uso de Código G.
